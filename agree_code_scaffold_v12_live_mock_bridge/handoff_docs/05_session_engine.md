Session Engine

Host sets
- category: eat / watch
- mood
- distance (eat)
- practical filters
- participants

Session rules
- same shared candidate pool for all participants
- different order per participant
- each user swipes independently
- timer: 2 minutes or 30-second quickie
- session ends when timer expires or winner becomes mathematically certain

Gestures
- right = like
- left = dislike
- up = really want
- down = never

Gesture weights
- really want = +3
- like = +2
- dislike = -1
- never = -3
- skip = 0

Anonymous intensity signal
- show subtle: 🔥 1 person really wants this
- never reveal identities

Result reveal
- winner only first
- show rating / distance or platform
- show taste match %
- show participant avatars
- actions:
  - Eat: Directions / Reserve / Order
  - Watch: Watch Trailer / Open Streaming Service / Add to Watchlist
  - Share
  - See Backup
